# Apex Interview Questions

### Beginner

- What is Apex?

Apex is Salesforce's Object Oriented Programming Language, which allows developers to execute backedn and Database operations to create custom business logic.

- What are the data types available in Apex?

Primitive: Blob, boolean, Date, Datetime, Decimal, Double, Id, Integer, Long, String, Time etc.

sObject: Standard, Custom, Generic sObject.

Collection: Lists, Sets, Maps

Enums: Season {WINTER, SPRING, SUMMER, FALL}

- What are Collections in Apex?

Collections are structured groups of elements that can hold primitives, sObjects, or other nested objects. it can be of Lists, Sets, Maps.


- Difference between List, Set, and Map?

List: It is ordered Collection that holds exact order in which elements are inserted, and can be accessed specific numerical Index.[ArrayList]
Set: It is Unodered collection of unique elements, without duplicate values.
Maps: Stores Data in Key-Value Pairs where key is unique and acts as index to hold corresponding value.

- What is an SObject?

(Salesforce Object) is a data Structure representing a single record or list of database data. It can be Standard, or custom and have page layout , serves as Primary way to interact with data in Apex.

- What are Wrapper Classes?

programmer-defined data type or container class that groups different data types or objects into a single unit.
