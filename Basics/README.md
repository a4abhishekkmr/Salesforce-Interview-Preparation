# Questions
### What is Salesforce
Salesforce is a cloud-based Customer Relationship Management (CRM) platform used to manage sales, service, marketing, and business processes.

### What is the difference between Standard Objects and Custom Objects?
Standard Objects: Account, Contact, Opportunity, Lead, etc.
Custom Objects: user Created objects to store custom business Data, eg. personal_loan__c.

### What is the difference between Role and Profile?
Profile controls object permissions, field permissions, tabs, apps, etc.
Role controls record-level access through Role heirarchy.

### What is the difference between Permission Sets and Profiles?
Profile = Base permissions for all users.
Permission Set = Permissions on top of Profile level.

### Order of Record-Level Access in Salesforce?
Organization Wide Defaults( OWD )
Role Hierarchy
Sharing Rules
Manual Sharing
Apex Sharing

### What is Organization Wide Defaults( OWD )?
Baseline record Access settings that determine the default visiblity of records.

### Difference between Sharing and Without Sharing?
With Sharing -> Enforces Sharing Rule
Without Sharing -> Ignores Sharing Rule.

## Apex
### What is Apex?
Apex is Salesforce's Object-oriented programming Language used to execute business logic in Salesforce.
